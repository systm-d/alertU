# alertu-settings

The [AlertU](https://github.com/systm-d/alertU) settings window: an egui/eframe
app for editing the whole configuration — remote, watched devices, delays,
sounds, camera and webhook — and pushing it to the daemon live.

Launched from the tray's "Open settings…" item, or standalone with
`--socket <path>`. Needs a running `alertu-daemon`.
